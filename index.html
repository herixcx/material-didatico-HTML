EXEMPLOS BÁSICOS DE HTML

Para nossos primeiros passos, mostraremos alguns exemplos básicos de HTML. Mas não se preocupe se usarmos tags que você ainda não conhece.

DOCUMENTOS HTML


Todos os documentos HTML devem começar com uma declaração de tipo de documento: < !DOCTYPE >.
O próprio documento HTML começa <html> e termina com </html>.
A parte visível do documento HTML está entre <body> e </body>.

Exemplo de Código HTML

                    <!DOCTYPE html>
                    <html lang="pt-BR">
                    <head>
                    <meta charset="UTF-8">
                    <title>Minha Página HTML</title>
                    </head>
                    <body>
                    <h1>Olá, mundo!</h1>
                    </body>
                    </html>

A DECLARAÇÃO <!DOCTYPE>


A <!DOCTYPE> declaração representa o tipo de documento e ajuda os navegadores a exibir as páginas da web corretamente.
Deve aparecer apenas uma vez, no topo da página (antes de qualquer tag HTML).
A <!DOCTYPE> declaração não diferencia maiúsculas de minúsculas.
A <!DOCTYPE> declaração para HTML5 é:

                        <!DOCTYPE html>

ELEMENTOS HTML ANINHADOS


Os elementos HTML podem ser aninhados (isso significa que os elementos podem conter outros elementos).
Todos os documentos HTML consistem em elementos HTML aninhados.
O exemplo a seguir contém quatro elementos HTML ( , <html> e ) : <body><h1><p>

Exemplo Código HTML:

                        <!DOCTYPE html>
                        <html>
                        <body>
                        
                        <h1>Primeiro Título.</h1>
                        <p>Primeiro parágrafo.</p>
                       
                        </body>
                        </html>

Explicando o exemplo:

➢ O elemento <html> é o elemento raiz e define todo o documento HTML. Possui uma tag inicial <html> e uma tag final </html>.
Então, dentro do elemento <html> existe um elemento <body>.
➢ O elemento <body> define o corpo do documento. Possui uma tag inicial <body> e uma tag final </body>.
Então, dentro do elemento <body> existem outros dois elementos: <h1> e <p>.
➢ O elemento <h1> define um título. Possui uma tag inicial <h1> e uma tag final </h1>.
➢ O elemento <p> define um parágrafo. Possui uma tag inicial <p> e uma tag final </p>.

✖ IMPORTANTE: NUNCA PULE A TAG FINAL ✖
Alguns elementos HTML serão exibidos corretamente, mesmo se você esquecer a tag final. No entanto, nunca confie nisso! Resultados inesperados e erros podem ocorrer se você esquecer a tag final!

● HTML NÃO DIFERENCIA MAIÚSUCLAS DE MINÚSCULAS

As tags HTML não diferenciam maiúsculas de minúsculas: <P> significa o mesmo que <p>. O padrão HTML não exige tags em minúsculas, mas recomenda-se letras minúsculas em HTML e exige letras minúsculas para tipos de documentos mais rígidos, como XHTML.

ATRIBUTO SRC


A tag <img> é usada para incorporar uma imagem em uma página HTML. O src atributo especifica o caminho para a imagem a ser exibida.
Existem duas maneiras de especificar o URL no src atributo:

➢ URL ABSOLUTA = Links para uma imagem externa hospedada em outro site. Exemplo: src="https://www.google.com/img/dog.png".
➢ URL RELATIVO = Links para uma imagem hospedada no site. Aqui, o URL não inclui o nome de domínio. Se o URL começar sem barra, será relativo à página atual. Exemplo: src="dog.png". Se o URL começar com uma barra, será relativo ao domínio. Exemplo: src="/img/dog.png".

ATRIBUTO LANG


Você deve sempre incluir o atributo lang dentro da tag <html>, para declarar o idioma da página web. O objetivo é ajudar os mecanismos de pesquisa e navegadores.
O exemplo a seguir especifica o inglês como idioma:

Exemplo Código HTML:

                        <!DOCTYPE html>
                        <html lang="en">
                        <body>
                        ...
                        </body>
                        </html>

Os códigos de país também podem ser adicionados ao código do idioma no atributo lang. Assim, os dois primeiros caracteres definem o idioma da página HTML e os dois últimos caracteres definem o país.
O exemplo a seguir especifica inglês como idioma e Estados Unidos como país:

Exemplo Código HTML:

                            <!DOCTYPE html>
                            <html lang="en-US">
                            <body>
                            ...
                            </body>
                            </html>

● COMENTÁRIOS HTML

Os comentários HTML não são exibidos no navegador, mas podem ajudar a documentar seu código-fonte HTML.



ETIQUETA DE COMENTÁRIO HTML


Você pode adicionar comentários à sua fonte HTML usando a seguinte sintaxe:

Exemplo Código HTML:

            <!-- ESCREVA SEUS COMENTÁRIOS AQUI -->

Observe que há um ponto de exclamação (!) na tag inicial, mas não na tag final. Os comentários não são exibidos pelo navegador, mas podem ajudar a documentar seu código-fonte HTML.



ADICIONE COMENTÁRIOS

Com comentários você pode colocar notificações e lembretes em seu código HTML:

Exemplo Código HTML:

            <!-- COMENTÁRIO AQUI -->

            <p>Esse é um parágrafo.</p>

            <!-- LEMBRAR DE ADICIONAR MAIS INFORMAÇÕES AQUI -->


OCULTAR CONTEÚDOS


Os comentários podem ser usados ​​para ocultar conteúdo.
Isso pode ser útil se você ocultar o conteúdo temporariamente:

LINKS HTML

Links são encontrados em quase todas as páginas da web. Os links permitem que os usuários cliquem de uma página para outra.



LINKS HTML - HIPERLINKS


Links HTML são hiperlinks.
Você pode clicar em um link e ir para outro documento.
Quando você move o mouse sobre um link, a seta do mouse se transforma em uma pequena mão.


LINKS HTML - SINTAXE


A tag HTML <a> define um hiperlink. Possui a seguinte sintaxe:

Exemplo Código HTML:

                <a href="url">link text</a>

O atributo mais importante da tag <a> é o atributo href, que indica o destino do link.
O texto do link é a parte que ficará visível ao leitor.
Clicar no texto do link enviará o leitor ao endereço URL especificado.



Exemplo Código HTML:

                <a href="https://www.google.com/">Visite Google.com!</a>

Exemplo:

Visite Google.com!

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:
➢ Um link não visitado está sublinhado e azul
➢ Um link visitado está sublinhado e roxo
➢ Um link ativo está sublinhado e vermelho



LINKS HTML - O ATRIBUTO DE DESTINO


Por padrão, a página vinculada será exibida na janela atual do navegador. Para alterar isso, você deve especificar outro destino para o link.
O atributo target especifica onde abrir o documento vinculado.
O atributo target pode ter um dos seguintes valores:

➢ _self- Padrão. Abre o documento na mesma janela/aba em que foi clicado
➢ _blank- Abre o documento em uma nova janela ou guia
➢ _parent- Abre o documento no quadro pai
➢ _top- Abre o documento em todo o corpo da janela


Exemplo Código HTML:

                   <a href="https://www.google.com/" target="_blank">Visite Google.com!</a>

Exemplo:

Visite Google.com!


URLS ABSOLUTOS X URLS RELATIVOS


Ambos os exemplos acima usam um URL absoluto (um endereço da web completo) no atributo href.
Um link local (um link para uma página dentro do mesmo site) é especificado com um URL relativo (sem a parte “https://www”):


Exemplo Código HTML:

SINTAXE DE IMAGENS HTML


A tag HTML <img> é usada para incorporar uma imagem em uma página da web.

As imagens não são tecnicamente inseridas em uma página da web; as imagens estão vinculadas a páginas da web. A tag <img> cria um espaço de retenção para a imagem referenciada.
A tag <img> está vazia, contém apenas atributos e não possui tag de fechamento.
A tag <img> tem dois atributos obrigatórios:

➢ src - Especifica o caminho para a imagem
➢ alt - Especifica um texto alternativo para a imagem

Exemplo Código HTML:

                    <img src="url" alt="alternatetext">


O ATRIBUTO SRC


O atributo obrigatório src especifica o caminho (URL) para a imagem.
NOTA: Quando uma página web carrega, é o navegador, naquele momento, que obtém a imagem de um servidor web e a insere na página. Portanto, certifique-se de que a imagem realmente permaneça no mesmo lugar em relação à página web, caso contrário seus visitantes receberão um ícone de link quebrado. O ícone do link quebrado e o texto alt serão mostrados se o navegador não conseguir encontrar a imagem

Exemplo Código HTML:

                    <img src="img/gaga1.webp" alt="Lady Gaga 1">


O ATRIBUTO ALTERNATIVO


O atributo obrigatório alt fornece um texto alternativo para uma imagem, se o usuário por algum motivo não puder visualizá-la (por causa de uma conexão lenta, um erro no atributo src ou se o usuário usar um leitor de tela).
O valor do atributo alt deve descrever a imagem:

Exemplo Código HTML:

                    <img src="img/gaga1.webp" alt="Lady Gaga 1">

Se um navegador não conseguir encontrar uma imagem, ele exibirá o valor do atributo alt:

Exemplo Código HTML:

                    <img src="wrongname.gif" alt="Lady Gaga 1">

FAVICON HTML

Um favicon é uma pequena imagem exibida ao lado do título da página na guia do navegador.


ADICIONANDO UM FAVICON EM HTML


Você pode usar qualquer imagem que desejar como favicon. Você também pode criar seu próprio favicon em sites como https://www.favicon.cc

DICA: Um favicon é uma imagem pequena, portanto deve ser uma imagem simples com alto contraste.

Para adicionar um favicon ao seu site, salve sua imagem de favicon no diretório raiz do seu servidor web ou crie uma pasta no diretório raiz chamada imagens e salve sua imagem de favicon nesta pasta. Um nome comum para uma imagem de favicon é “favicon.ico”.
Em seguida, adicione uma tag <link> ao seu arquivo “index.html”, após a tag <title>, assim:

Exemplo Código HTML:

                            <!DOCTYPE html>
                            <html>
                            <head>
                                <title>Título da Minha Página</title>
                                <link rel="icon" type="image/x-icon" href="/img/favicon.ico">
                            </head>
                            <body>

                            <h1>Esse é um Título</h1>
                            <p>Esse é um parágrafo.</p>

                            </body>
                            </html>;

Agora salve o arquivo “index.html” e recarregue-o em seu navegador. A guia do seu navegador agora deve exibir a imagem do seu favicon à esquerda do título da página.
